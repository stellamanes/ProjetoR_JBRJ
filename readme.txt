Este eh o repositorio da disciplina ENBT 2019 ministrada por Andrea Sanchez, Sara Mortara
e Diogo.


O ideal eh que a pasta dos projetos esteja padronizada e organizada de maneira que qualquer um entenda.

Os arquivos estao organizados da seguinte forma:
Na pasta R so vai ter os scripts do R
Na pasta Data vamos botar os dados brutos usados pra rodar as analises
Na pasta Results vai ter todos os resultados salvos
Na pasta docs tem os textos e pdfs que vao ser usados
