O ideal é que a pasta dos projetos esteja padronizada e organizada de maneira que qualquer um entenda.


Os arquivos estão organizados da seguinte forma:
Na pasta R só vai ter os scripts do R
Na pasta Data vamos botar os dados brutos usados pra rodar as analises
Na pasta Results vai ter todos os resultados salvos
e Na pasta docs tem os textos e pdfs que vao ser usados
